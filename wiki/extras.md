Ubuntu extras
=================

Extras para desktops ubuntu

1 Instalação
--------------------------------------------------------

1.1 Atualiza

`sudo apt update`

1.2 Instala

`sudo apt install ubuntu-restricted-addons -y`

`sudo apt install ubuntu-restricted-extras -y`

1.3 Atualiza

`sudo apt update ; apt list --upgradable ; sudo apt upgrade -y`
