Diodon
========================================================

1 O que é?
--------------------------------------------------------

É um software gerenciador de área de transferência.

2 Instalação
--------------------------------------------------------

`sudo add-apt-repository ppa:diodon-team/stable`

`sudo apt-get update`

`sudo apt-get install diodon`

3 Desinstalação
--------------------------------------------------------

`sudo apt-get remove diodon unity-scope-diodon --auto-remove`
