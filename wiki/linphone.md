Linphone
=================

Programa para realizar ligações por VOIP.

1 Instalação
--------------------------------------------------------

1.1 Atualiza

`sudo apt update`

1.2 Instala

`sudo apt install linphone -y`

1.3 Atualiza

`sudo apt update ; apt list --upgradable ; sudo apt upgrade -y`
