Usuarios Linux
========================================

1 Grupos
---------------------------------------

1.1 Criar um grupo

`sudo addgroup nomedogrupo`

1.2 Incluir um usuario em um grupo.

`sudo adduser nomedousuario nomedogrupo`

1.3 Incluir usuario no grupo sudo (deve ser executado como root).

`usermod -a -G sudo nomedousuario`
